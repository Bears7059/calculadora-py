# Calculadora Web

Esta es una calculadora hecha con **HTML, CSS y JavaScript**.

## Funcionalidades
- Suma, resta, multiplicación y división  
- Interfaz sencilla  
- Funciona directamente desde el navegador

##  Cómo ejecutar
Solo abre el archivo **calculadora.html** en tu navegador:

Cómo ejecutar la calculadora

Clona el repositorio:

git clone https://github.com/tu-usuario/calculadora-py.git


Entra en la carpeta:

cd calculadora-py


Abre el archivo:

index.html


(Opcional) Si usas VS Code, abre con Live Server:

Clic derecho → “Open with Live Server”
